# JCB Stock Manager

## Tech Stack

- React
- Vite
- Tailwind
- shadcn/ui
- Supabase

## Features

- View all products
- Search by part number or name
- Add product
- Edit product
- Delete product
- Increase stock
- Decrease stock
- Highlight low stock

## Product Schema

- id
- part_number
- name
- quantity
- minimum_quantity
- created_at

## UI

Single-page dashboard with:
- Search bar
- Add Product button
- Product table
- + Stock
- - Stock
- Edit
- Delete
